Сервис загрузки и обработки фотографии
=================

Простой сервис загрузки и обработки фотографии на Django.

Пользователь открывает страницу и/или в определенное понятное место на странице перетаскивает или выбирает файл. 
Загрузить можно только картинку (видео, текст и прочее, нельзя).
Ограничений на загрузку нет. При загрузке на стороне сервера файл сжимается.
На стороне сервера есть модель, где храниться путь к файлу
Обработка загрузки осуществляется методами DRF.

сервер: django, drf
клиент: dropzone.js
